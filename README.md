# Flask-exam

Веб-приложение для проведения психологического тестирования в школе. Разработано на Flask.

## Цель

Создать сайт-помощник для проведения типовых психологических тестов с разделением для учеников и преподавателей.

## Функционал

**Для учеников:**
- Вход по логину
- Прохождение теста
- Автоматическая проверка результатов

**Для преподавателей (пароль: 123):**
- Загрузка вопросов из CSV файла
- Добавление учеников в базу данных
- Просмотр результатов всех учеников

## Технологии

- Python 3
- Flask
- SQLite3
- HTML/CSS

## Установка

### 1. Скачайте нужную версию

Выберите папку с вашим классом:

- **Для 7-9 классов** — скачайте папку `7-9/`
- **Для 10-11 классов** — скачайте папку `10-11/`

### 2. Перейдите в папку проекта

cd 7-9   # или cd 10-11
### 3. Создайте виртуальное окружение
bash
python -m venv venv
### 4. Активируйте окружение
bash
# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
### 5. Установите зависимости
bash
pip install -r requirements.txt
### 6. Запустите приложение
bash
python serv.py
